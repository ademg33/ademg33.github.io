RENDER. If you want to skip the trial and error altogether, you can make a high-quality artificial lens flare in the post-processing phase by following these easy steps. 1. Open your photo in Photoshop. 2. Create a new layer. Go to Edit › Fill, and fill it with 50% gray. 3. Set the blending mode to Overlay. 4. Go to Filter › Render › Lens flare. Select which type of lens flare you want to emulate. 5. Position your lens flare, then click OK.
